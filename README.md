## Overview

This repository presents my professional contribution to the **ACTS Management System**, developed at C-DAC Bangalore.

The ACTS Management System is an enterprise-level web-based ERP platform designed to manage the complete lifecycle of PG-Diploma students, including registration, verification, document management, and administrative workflows.

This portfolio is shared strictly for evaluation purposes. Due to organizational confidentiality policies, the complete production source code cannot be publicly disclosed. This repository contains screenshots, architectural overview, and representative code samples demonstrating my technical contributions.

## System Architecture Overview

The ACTS Management System follows a client-server architecture:

- React-based frontend for dynamic and responsive UI
- RESTful APIs built using Node.js & Express
- MySQL relational database for structured data storage
- Secure document storage with controlled directory structure
- Role-based access control for multiple dashboards
- Dockerized services deployed behind Nginx reverse proxy
- VM-based production hosting environment
